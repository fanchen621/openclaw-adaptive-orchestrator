# 🧠 OpenClaw 自适应编排器 (Adaptive Orchestrator)

智能自适应任务编排技能，让 OpenClaw 面对多任务或复杂任务时，自动拆解、排序、并行执行，并从失败中学习。

## 核心能力

| 模块 | 功能 | 实现 |
|------|------|------|
| 📋 规划器 | 任务拆解、依赖分析、优先级排序 | `sessions_spawn` 并行子代理 |
| 🗜️ 监控器 | Token 消耗监控、上下文自动压缩 | `session_status` + 摘要汇总 |
| 🧠 经验库 | 失败模式 → 成功方案持久化记忆 | JSON 文件 + 关键词匹配 |

## 安装

将 `SKILL.md` 复制到你的 OpenClaw skills 目录：

```bash
cp SKILL.md ~/.openclaw/skills/adaptive-orchestrator/SKILL.md
```

## 经验库

初始包含 3 条内置经验：
- **腾讯云 COS 故障** → 自动切换 CDN 直链/备用站点
- **API 429 限流** → 指数退避重试 (2s, 4s, 8s, 16s)
- **上下文过长** → 压缩为精简状态报告

经验库会随着使用自动扩充——每次成功解决新类型的错误，都会存入经验库供下次复用。

## 使用示例

```
用户: "帮我检查三个 COS 站点，挂了就换直链，结果整理成表格"

→ 规划器拆解: 并行检查3站点 → 条件切换链接 → 整理表格
→ 执行中遇到 COS 403 → 经验库命中 → 自动切直链 → 成功
→ 输出完成报告
```
